# Business Operator

Tony V5 can draft business communication locally.

## Drafts

- Client update message
- Delivery note
- Progress update
- Proposal outline
- Follow-up message
- Delay/apology message
- Daily and project reports

Tony does not send emails, WhatsApp messages, SMS, invoices, or client messages automatically.

## Local Storage

Drafts, reports, prompts, daily plans, and tasks are saved locally in SQLite under `tony/logs/`.

Tony refuses to save passwords, API keys, private secrets, tokens, or payment information.
