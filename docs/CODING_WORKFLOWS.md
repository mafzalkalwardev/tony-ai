# Coding Workflows

Tony V5 includes safe predefined workflows.

## Analyze Project

- Detect project type
- Check Git status
- List scripts and commands
- Generate a local summary

## Debug Project

- Analyze project
- Recommend test/run commands
- Explain error text
- Suggest fix
- Stop before running or editing until approval

## GitHub Delivery

- Show repo status
- Generate commit message draft
- Generate delivery note
- Stop before commit, push, or PR creation

## Client Update

- Summarize work context
- Generate editable client message draft
- Save locally only

## Prompt Generation

Tony can generate prompts for Codex, Cline, Cursor, or other coding tools. Prompts are sanitized for obvious secret markers.
