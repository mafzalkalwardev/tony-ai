# Roadmap

## V1: Text Agent

- PyQt6 desktop UI
- Typed commands
- Local Ollama brain
- Git and shell tools
- Safety approval system
- SQLite memory and logs

## V2: Voice Wake Phrase

- Push-to-talk voice input
- Local faster-whisper speech-to-text
- Local pyttsx3 text-to-speech startup greeting
- Graceful microphone/model fallback
- Wake phrase: `Wake up Tony, Daddy's Home`
- Always-listening wake phrase detection later

## V3: Laptop Operator

- App/window awareness
- Safer UI automation with pywinauto or pyautogui
- Explicit permission before clicks, typing, or browser automation
- Reusable workflows for local development

## V4: GitHub and Business Automation

- GitHub CLI workflows
- PR summaries and issue triage
- Client message drafting
- Business task checklists
- No automatic sending without approval

## V5: Advanced Memory and Workflows

- Project-specific long-term memory
- Routine detection
- Multi-step plans
- Better audit logs and rollback points
- Local workflow templates
