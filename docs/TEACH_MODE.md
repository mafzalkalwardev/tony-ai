# Teach Mode

Teach Mode lets Tony learn a workflow locally after approval.

## Commands

```text
Tony watch me
Tony mujhe dekh ke seekho
Tony workflow save karo
Tony workflows dikhao
Tony last workflow replay karo
```

## What Is Recorded

- Safe action metadata
- Window title
- Timestamps
- Notes
- Optional local screenshots at key moments when approved

Tony does not intentionally save passwords, OTPs, API keys, secrets, banking/payment details, or private message sending actions.

## Replay

Replay always requires approval because it may control mouse and keyboard. Dry run and preview are safe and should be used before replay.
